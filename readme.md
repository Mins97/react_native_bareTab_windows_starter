# 맥으로 시작할 수 있는 스타터 킷입니다.
```
 sh stater.sh 를 하면 바탕화면에 새 프로젝트를 만들어줍니다.
 환경설정까지!
 ROOT는 메인 폴더입니다.
 AppName은 앱의 이름입니다.
 MacName는 맥의 계정 이름입니다.
```
# 시작하기
```
cd ~/Desktop &&
git clone https://github.com/Mins97/react_native_mac_starter.git starter &&
cd ~/Desktop/starter &&
cd testapp &&
sh init.sh
```
## 실행 명령어
```
react-native run-android
```
 또는
```
react-native run-ios
```
